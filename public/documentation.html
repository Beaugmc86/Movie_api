myFlix API Documentation

A server-side component of a "movies" web application.
The web application will provide users with access to information
about different movies, directors, and genres. Users will be able
to sign up, update their personal information, and create a list of
their favorite movies.


Request URL HTTP Method Request body data format Response body data format
Get list of all movies /movies GET None An array of JSON objects holding data about all movies within the database
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about the requested movie like this:

{'title': 'Dawn of the Dead',
'director': 'George A. Romero',
'genre': 'Horror',
'release-year': '1978'}
Get list of movies by genre /movies/genre/[genre] GET None An array of JSON objects holding data about all the movies by requested genre
Get list of movies by director /movies/directors/[director] GET None An array of JSON objects holding data about all the movies by requested director
Add a user /users POST A JSON object holding data about the user to add, structured like:

{'Name': '[Beau]',
'Password': '[Password]',
'Email': '[email]' } 		A JSON object holding data about the user that was added, including their assigned id:

{ 'id': '###',
'Name': [Beau],
'Password': '[Password]',
'Email': '[email]',}
Update name info /users/[name] PUT A JSON object holding data about the user which needs to be updated, example:

{ 'name': 'req.body.name',
'Password': 'req.body.password',
'Email': 'req.body.email' } 		A JSON object holding data about the updated user info, example:

{ 'id': '###',
'name': '[new name]',
'Password': '[password]',
'Email': '[email]'}
Remove a user /users/[id] DELETE None A text message indicating whether user was deregistered
Add a movie object to a user's favorites list /users/[id]/[Title] POST None A text message indicating whether the movie was successfully added to the user's favorited movies
Remove a movie from a user's favorites list /users/[id]/[Title] DELETE None A text message indicating whether the movie was successfully removed from the user's favorited movies
